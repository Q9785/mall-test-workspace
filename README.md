# mall商城全链路测试实战作品集

> 基于开源电商项目 mall 的完整测试实战  
> 测试周期：2026.05 - 2026.06（约1.5个月）
> 测试环境：Windows + MySQL + Redis + IDEA

##  项目简介

mall 是一款基于 Spring Boot + Vue 的开源电商系统，包含 B端后台管理（mall-admin）和 C端前台商城（mall-portal）。

本次测试聚焦 **B端商品/订单管理** 和 **C端登录/购物车/下单/支付/秒杀** 核心交易链路，覆盖功能、接口、自动化、性能、安全、弱网、兼容性等测试维度。

**核心成果**：
- 设计测试用例 **200+ 条**
- 发现有效 Bug **15+ 个**
- P0 级核心流程覆盖率 **100%**
