# 安全、弱网、兼容性测试专项说明

| 测试点 | 结论 |
|---|---|
| SQL注入防护 |  系统具备防注入能力  |
| 明文密码传输 | ❌ 密码明文传输，建议启用HTTPS及前端加密 |
| 越权防护 |  前后台Token隔离有效，无法跨用户访问 |
| Token鉴权 |  过期/无效Token无法通过认证 |
| 价格篡改防护 | ❌ 后端未校验价格合法性，负数价格可写入数据库 |

| 测试维度 | 结论 |
